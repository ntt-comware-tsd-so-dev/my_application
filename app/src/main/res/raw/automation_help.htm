Step 1. Add a new Geofence Location or a Beacon to AMAP app.

 

Step 2. To add a new Geofence go to navigational window then click on GeofenceLocations and click on + button. Enter the name of the Geofence and select default current location or Edit location. Change the Radius (if needed) from default of 50.0m.

 

Step 3. Similarly to add a new Beacon go to navigational window click on Beacons and click on + button. If there are Beacons available a list of Beacons are displayed.

 

Step 4. Add a new Action. To create a new Action go to Navigational window and click on Device actions and click on + button.

 

Step 5. Go to Automations in the navigational window then click on + button. Enter the name and select either Geofence Locations or a Beacon. Select if you want to fire the event while entering or exiting. Finally select the actions associate with this Automation and click on Save.
