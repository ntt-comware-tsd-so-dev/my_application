Device Automations

Automations can be used to control your devices based on your location. Geofences and Beacons can be set up to perform actions when you approach or leave a specified location.

A Geofence uses your mobile device's GPS to determine when you have entered or left a certain area on the map.

Beacons use your mobile device's Bluetooth radio to determine how close you are to a particular beacon.

Actions in AMAP consist of a device, a property and a value. When the action is executed, the selected property of the selected device will be set to the provided value.

Automations represent the pairing of a Geofence or Beacon event with an Action, allowing devices to be manipulated based on your location.

Events

An Event is something that happens that causes Actions to be taken. Currently, AMAP supports two types of events: Geofences and Beacons. Both events are based on the physical location of the mobile device. Geofences use the mobile device's GPS location to determine position, while Beacons use the physical proximity to a Beacon as a source of the event.

Geofences

A Geofence is basically a radius around a point on the map. Events can be triggered if the mobile device enters or leaves the geofence area (e.g. comes within or leaves the circle defined by the point and radius). AMAP supports up to 5 simultaneous geofences. To create a geofence:

  1. In the main menu, select Geofence Locations
  2. Tap the '+' button to create a new Geofence
  3. A map will be presented to you with a pin representing the location of your geofence and a circle around it representing the radius
  4. Scroll the map around to change the center location of the geofence
  5. Slide the slider below the map to change the radius of the geofence
  6. When the location and radius are correct, tap the edit box to name your geofence and hit Save

Beacons

A Beacon is similar to a Geofence in that it is used to help determine the proximity of the mobile device, but the location and radius of a Beacon is based on the physical location of the Beacon and its strength. In order to use a Beacon with AMAP, a BLE beacon must be purchased that supports either the iBeacon or Eddystone beacon standards. AMAP supports up to 5 simultaneous Beacons. To set up a Beacon within AMAP:

  1. In the main menu, select Beacons
  2. Tap the '+' button to add a new beacon
  3. Once the app scans for and discovers your beacon, select it and choose a name for it

Actions

Actions are used to control your devices when an event happens. An action consists of a device, a property and a value. When the action is run, the property of the device is set to the specified value. This can be used to turn on or off devices, set light levels of dimmable bulbs, or control your devices in just about any way.

To create an Action:

  1. In the main menu, select Device Actions
  2. Here you will see a list of your registered devices. Find the device you wish to control, and tap the '+' button
  3. Choose a name for your action and enter it in the Action Name field
  4. From the drop-down list, select the property you wish to modify with this action
  5. Once the property has been selected, enter the value the Action should set the property to in the edit field
  6. Tap Save to save your action. The action may now be used in automations.

Automations

Automations is where everything is tied together. An Automation ties an Event (Geofence or Beacon) to an Action. AMAP supports up to 10 simultaneous Automations.

To create a new Automation, first make sure you have at least one Event and one Action already defined, and then:

  1. From the main menu, select Automations
  2. Tap the '+' button to create a new Automation
  3. Choose a name for your automation and enter it into the Automation Name field
  4. Select either Geofence Locations or Beacons, depending on which trigger you would like to use
  5. From the drop-down list, choose the Geofence Location or Beacon you wish to use as the event source
  6. Next to Perform these actions, tap the '+' button
  7. From the list of defined actions, check the boxes next to the actions you would like performed in response to this event. Tap Done when finished.
  8. Tap Done to create your automation
Your new automation will now appear in the list of automations. The automation may be enabled or disabled from this screen by sliding the switch.
